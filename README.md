
# Network Sniffer - CodeAlpha Cyber Security Internship

## Description
A Python-based network packet sniffer built using Scapy library. 
This tool captures and analyzes network traffic in real-time.

## Features
- Captures live network packets
- Displays Source & Destination IP
- Shows Protocol (TCP/UDP)
- Shows Port numbers
- Displays TCP Flags
- Captures HTTP payload data

## Requirements
- Python 3
- Scapy library
- Linux (Kali Linux recommended)
- Root/sudo access

## Installation
```bash
sudo apt install python3-scapy -y
```

## Usage
```bash
sudo python3 network_sniffer.py
```

## Sample Output
- Source IP → Destination IP
- Protocol type (TCP/UDP)
- Port numbers
- Packet flags
